# Demo Banking Automation Testing Project

## Project Overview
This project automates functional test scenarios of a demo banking web application using Selenium WebDriver with Java and Cucumber (BDD). The framework is designed based on Page Object Model (POM) principles.

## Tools & Technologies
- Java
- Selenium WebDriver
- Cucumber (BDD)
- TestNG
- Maven
- Git & GitHub

## Automated Scenarios
- User Login
- Account Creation
- Fund Transfer
- Balance Enquiry
- Logout

## Framework Design
- BDD framework using Cucumber
- Feature files written in Gherkin language
- Step Definition classes
- Page Object Model (POM)

## How to Execute
1. Clone the repository
2. Import the Maven project
3. Run the TestRunner class
4. View execution results in the console

## Key Benefits
- Business-readable test cases using BDD
- Modular and reusable framework
- Suitable for real-time banking automation scenarios
